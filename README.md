SkillQuest is a mobile educational platform built with **React Native**. It helps users learn, practice, and master new skills through structured video courses, interactive quizzes, and a live AI chatbot. The app is designed to provide a dynamic and engaging learning experience for anyone interested in tech.

-----

### **Features**

  * Diverse course catalog covering topics like **C++, Java, Python, Data Science**, and web development.
  * Interactive quizzes to reinforce knowledge and track progress.
  * An AI chatbot, **Echo**, for instant support and guidance.
  * Personalized profile management to monitor achievements and learning stats.
  * A clean and intuitive UI designed for a seamless learning experience.

-----

### **Tech Stack**

**Frontend:**

  * React Native
  * Expo
  * React Navigation

**Backend:**

  * Node.js
  * Express.js

-----

### **Installation**

1.  **Clone the repository:**

    ```bash
    git clone <your-repo-url>
    cd SkillQuest
    ```

2.  **Install frontend dependencies:**

    ```bash
    cd frontend
    npm install
    ```

3.  **Install backend dependencies:**

    ```bash
    cd ../backend
    npm install
    ```

4.  **Run the app:**

    ```bash
    # Start the backend server
    node server.js

    # Start the frontend app
    cd ../frontend
    npm start
    ```
